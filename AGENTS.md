# Mobiliza Inteligente

Platform for political campaign intelligence and training.

## Project Structure

Static HTML website with pages for courses, instructors, campaign methods, territorial mobilization, news, certifications, and more.

## Skills Available

This project includes Stitch skills for AI-powered design workflows:

### Design Skills
- **stitch-generate-design** - Generate/ edit screens from text or images
- **stitch-code-to-design** - Convert HTML/CSS code to Stitch designs
- **stitch-manage-design-system** - Create and apply design systems
- **stitch-extract-design-md** - Extract design tokens from source code
- **stitch-extract-static-html** - Capture self-contained HTML snapshots
- **stitch-upload-to-stitch** - Upload assets to Stitch projects

### Build Skills
- **stitch-react-components** - Convert designs to React components
- **stitch-remotion** - Generate walkthrough videos from designs
- **stitch-shadcn-ui** - Integrate shadcn/ui components

### Utility Skills
- **stitch-design-md** - Generate DESIGN.md files
- **stitch-enhance-prompt** - Polish UI prompts before generation
- **stitch-loop** - Iterative multi-page site generation
- **stitch-taste-design** - Enforce premium design standards

## Setup

1. Get your Stitch API key from https://stitch.withgoogle.com/settings
2. Add it to `opencode.json` under `mcp.stitch.headers.X-Goog-Api-Key`
3. Skills are auto-discovered from `.opencode/skills/`
